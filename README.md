
# **Trabalho de Desenvolvimento de uma lib reutilizavél em Projeto de Sistemas**

Uma biblioteca(lib) é um conjunto de funções desenvolvidas de forma genéricas, com o intuíto de serem distribuidas
e reutilizadas em outros sistemas e por outros programadores.


## **Etapas**

1. Desenvolvimento de uma biblioteca de criação pessoas e de endereços;
2. Desenvolvimento de uma série de testes, a fim de testar as funçoes da lib;
3. Subir o repositório no github; 
4. Subir a lib no PiPy;
